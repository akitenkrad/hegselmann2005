# Use cases

This project reproduces the continuous opinion-dynamics model of Hegselmann & Krause (2005), generalizing the 2002 HK bounded-confidence model along the axis of *which average* aggregates opinions.

## What you can do

1. **Observe the three-phase transition.** Run a single simulation and watch how the confidence radius ε controls the outcome: small ε leaves many opinion clusters (plurality), intermediate ε produces two camps (polarization), and large ε yields consensus. See [CLI → run](cli.md) and [Visualization](visualization.md).

   ```bash
   cargo run --release -- run --n 625 --eps 0.05 --mean A --seed 42   # plurality
   cargo run --release -- run --n 625 --eps 0.15 --mean A --seed 42   # polarization
   cargo run --release -- run --n 625 --eps 0.25 --mean A --seed 42   # consensus
   uv run hegselmann-tools visualize
   ```

2. **Compare averaging operators.** Switch the `--mean` flag among A / G / H / P_p / R and observe how the choice of average shifts the consensus brink, the (a)symmetry of polarization, and the final mean opinion. Harmonic (H) gives asymmetric polarization (extreme pull toward small values); P_100 and R have no polarization region and head straight to consensus.

3. **Sweep ε and estimate the consensus brink.** Run `sweep` to scan ε across all five means and aggregate the occupied-class count and consensus brink ε*. The paper's ordering is `ε*(R) < ε*(P_100) < ε*(A) < ε*(G) ≈ ε*(P_0.01) < ε*(H)`. See [CLI → sweep](cli.md) and [Visualization → sweep](visualization.md).

   ```bash
   cargo run --release -- sweep --eps-min 0.0 --eps-max 0.40 --eps-step 0.01 \
       --means A,G,H,P0.01,P100,R --n 625 --runs 50 --seed 42
   uv run hegselmann-tools visualize-sweep
   ```

4. **Study convergence speed.** A/G/H/P stabilize quickly (often under 20 steps), while the random mean R can remain unstable even after thousands of steps. The `metrics.csv` `max_delta` column tracks the per-step maximum opinion change.

5. **Reproduce the paper's headline figures in one command.** Run `reproduce` to render the operator-comparison grid, the arithmetic three-regime transition, and the ε phase diagram, each checked against the paper's expected per-operator regime. See [Visualization → reproduce](visualization.md).

   ```bash
   uv run hegselmann-tools reproduce            # full paper-faithful run
   uv run hegselmann-tools reproduce --quick    # fast smoke run
   ```
