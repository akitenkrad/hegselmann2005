# ユースケース

本プロジェクトは Hegselmann & Krause (2005) の連続意見力学モデルを再現する．2002 年 HK 有界信頼モデルを「どの平均で意見を集約するか」という軸で一般化したものである．

## できること

1. **三相転移の観測．** 単一シミュレーションを実行し，信頼幅 ε が結果を支配する様子を見る: 小さな ε は多数の意見クラスタ (多元) を残し，中程度の ε は 2 陣営 (分極) を生み，大きな ε は合意に至る．[CLI → run](cli.ja.md) と [可視化](visualization.ja.md) を参照．

   ```bash
   cargo run --release -- run --n 625 --eps 0.05 --mean A --seed 42   # 多元
   cargo run --release -- run --n 625 --eps 0.15 --mean A --seed 42   # 分極
   cargo run --release -- run --n 625 --eps 0.25 --mean A --seed 42   # 合意
   uv run hegselmann-tools visualize
   ```

2. **平均演算子の比較．** `--mean` フラグを A / G / H / P_p / R で切り替え，平均の選択が合意ブリンク・分極の (非) 対称性・最終平均意見をどう変えるかを観測する．調和平均 (H) は非対称分極を生む (小さい値への極端な引力)．P_100 と R には分極領域がなく直接合意へ向かう．

3. **ε 走査と合意ブリンク推定．** `sweep` で全 5 種の平均について ε を走査し，占有クラス数と合意ブリンク ε* を集計する．論文の順序は `ε*(R) < ε*(P_100) < ε*(A) < ε*(G) ≈ ε*(P_0.01) < ε*(H)`．[CLI → sweep](cli.ja.md) と [可視化 → sweep](visualization.ja.md) を参照．

   ```bash
   cargo run --release -- sweep --eps-min 0.0 --eps-max 0.40 --eps-step 0.01 \
       --means A,G,H,P0.01,P100,R --n 625 --runs 50 --seed 42
   uv run hegselmann-tools visualize-sweep
   ```

4. **収束速度の調査．** A/G/H/P は速やかに安定する (しばしば 20 ステップ未満) のに対し，ランダム平均 R は数千ステップ後でも不安定なことがある．`metrics.csv` の `max_delta` 列がステップごとの最大意見変化量を記録する．

5. **論文の主要 Figure を 1 コマンドで再現．** `reproduce` を実行すると，演算子比較グリッド・算術平均の三相転移・ε 相図を描き，それぞれを論文の期待相と突き合わせる．[可視化 → reproduce](visualization.ja.md) を参照．

   ```bash
   uv run hegselmann-tools reproduce            # 論文値でフル再現
   uv run hegselmann-tools reproduce --quick    # 高速スモーク実行
   ```
